## Requirements
- Add checkbox to each transaction row
- "Select All" / "Deselect All" buttons
- Bulk action toolbar appears when items selected
- Bulk operations:
  - Bulk categorize
  - Bulk tag
  - Bulk delete (with confirmation)
- Show count of selected items
- Maintain selection across page navigation

## Implementation Plan
- Introduce front-end selection state with master checkbox, bulk toolbar, and count display.

- Provide confirmation flows and post-action refresh to keep UI in sync.
