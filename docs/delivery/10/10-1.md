## Description

Provide a JSON endpoint that computes core financial metrics over the past 30 days and returns a composite health score. The API powers the dashboard UI and future integrations.

## Requirements

- Calculate income, expenses, net income over trailing 30 days (exclude transfer categories).
- Compute savings rate `net_income / income`.
- Return total balance across accounts and active budget count.
- Provide top 5 spending categories with totals.
- Derive health score (0-100) from savings rate, balance, budgets, and net income.

## Implementation Plan

1. Create `dashboard` blueprint with `/dashboard/api/health`.
2. Write SQL queries for each metric using SQLite joins.
3. Implement scoring rubric with explicit weights.
4. Handle error cases gracefully (return JSON error).
