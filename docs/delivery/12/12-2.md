## Requirements

### Functional Requirements
1. Add bcrypt to requirements.txt
2. Create password utility service with:
   - `hash_password(password)` - Hash password with bcrypt
   - `verify_password(password, password_hash)` - Verify password against hash
   - `generate_salt()` - Generate encryption salt
   - `validate_password_strength(password)` - Check password requirements
3. Add authentication method to User model:
   - `authenticate(username, password)` - Verify credentials and return user
4. Password requirements:
   - Minimum 12 characters
   - At least one uppercase letter
   - At least one lowercase letter
   - At least one number
   - At least one special character
