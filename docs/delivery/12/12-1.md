## Requirements

### Functional Requirements
1. Create `users` table with:
   - id (PRIMARY KEY)
   - username (UNIQUE, NOT NULL)
   - email (UNIQUE, NOT NULL)
   - password_hash (NOT NULL)
   - encryption_salt (NOT NULL)
   - encrypted_db_key (for future SQLCipher support)
   - is_active (BOOLEAN, DEFAULT 1)
   - created_at (TIMESTAMP)
   - last_login (TIMESTAMP)

2. Create `user_sessions` table with:
   - id (PRIMARY KEY)
   - user_id (FOREIGN KEY to users)
   - session_token (UNIQUE, NOT NULL)
   - expires_at (TIMESTAMP, NOT NULL)
   - created_at (TIMESTAMP)

3. Create User model class with methods:
   - `create(username, email, password)` - Create new user
   - `get_by_id(user_id)` - Get user by ID
   - `get_by_username(username)` - Get user by username
   - `get_by_email(email)` - Get user by email
   - `update_last_login(user_id)` - Update last login timestamp
   - `deactivate(user_id)` - Deactivate user account

4. Create migration script `migrate_add_users.py`
